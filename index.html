ذكاء اصطناعي متقدم

نظام ذكي يعتمد على تقنيات IoT والذكاء الاصطناعي لإدارة منظومات الطاقة الشمسية بكفاءة عالية

جمع وتحليل البيانات

جمع بيانات حية من الحساسات (الإضاءة، الحرارة، الرطوبة، الفولت) مع بيانات الطقس الخارجية بنظام انترنت الاشياء

نمذجة تنبؤية دقيقة

معالجة البيانات لاتخاذ قرارات تشغيلية ذكية وتوقعات دقيقة لأفضل استفادة من الطاقة المتاحة

الأهداف الرئيسية

تحسين الإنتاج

زيادة كفاءة التوليد عبر إدارة توجيه الألواح (Solar Tracking) بدقة.

إدارة الأحمال

توقع قدرة النظام واتخاذ قرارات ذكية لتشغيل أو تأجيل الأحمال الكبيرة.

مراقبة كاملة

عرض معلومات مفهومة (حالة الشحن، التنبؤات) عبر تطبيق جوال سهل.

المكونات والتقنيات

الأجهزة الطرفية (IoT)

  • حساسات شدة الإضاءة (LDR)
  • حساسات الحرارة (DHT11)
  • وحدة ESP8266 والـ Servo

الخادم والمعالجة

  • خادم سحابي لتخزين البيانات
  • نماذج ذكاء اصطناعي تنبؤية
  • ربط مع API طقس عالمي

واجهة المستخدم

  • تطبيق فلاتر (Flutter)
  • رسوم بيانية تفاعلية
  • تنبيهات وقت حقيقي

آلية عمل النظام

جمع البيانات

الحساسات تقرأ المعطيات (فولطية، تيار، حرارة، إضاءة) دورياً وترسلها لوحدة الإرسال.

الإرسال السحابي

وحدة ESP8266 تبعث البيانات عبر الإنترنت إلى الخادم السحابي للمعالجة.

المعالجة والذكاء

تصفية البيانات، استدعاء بيانات الطقس، وحساب قدرة الشحن المتوقعة (SOC).

اتخاذ القرار

تفعيل متعقب الشمس، إرسال أوامر للأحمال، وعرض التنبؤات للمستخدم.

المراقبة والتحكم

متابعة النظام لحظياً عبر التطبيق وإمكانية التحكم اليدوي عند الحاجة.

الصيانة والتنبيهات

اكتشاف الأعطال مبكراً وإرسال إشعارات للصيانة أو تنظيف الألواح.

مبتكرين الفكرة

م.غمدان نشوان

مهندس برمجيات ومبتكر تقني يمني، رائد في مجال حلول الأنظمة الذكية والتحول الرقمي. مواقع وتطبيقات ويب حديثة، سريعة، وقابلة للتوسع باستخدام أحدث تقنيات الويب.

م.محمد احمد الزوعري

مختص تكنولوجيا معلومات ومبتكر تقني يمني، أعمل على إدارة وتطوير الأنظمة التقنية، البنية التحتية الرقمية، وحلول التحول الرقمي باستخدام أحدث التقنيات.

م.هشام قاسم صغير

مهندس برمجيات يمني مختص بتطوير الويب، أعمل على تصميم وبناء مواقع وتطبيقات ويب حديثة، سريعة، وقابلة للتوسع باستخدام أحدث تقنيات الويب. تقنيات الويب.
